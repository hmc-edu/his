# 01 · 环境准备

> 杭州医学院 门诊 HIS 系统 · 实训文档

本文档面向**完全没有搭过开发环境**的同学。按章节顺序操作即可，每一步都列出了：要装什么、装在哪、怎么验证装好了。

---

## 0. 总览：要装哪些东西

| 软件 | 版本 | 用途 |
| --- | --- | --- |
| **JDK 17** | 17.x（不能更低，也不必更高） | 编译/运行后端 |
| **Node.js** | 18.x 或 20.x | 编译/运行前端 |

| **Git** | 2.x | 版本控制工具 |
| **IntelliJ IDEA**（社区版） | 2023+ | 后端开发 IDE |
| **VS Code** | 任意 | 前端开发 IDE |
| **Docker Desktop**（可选） | 任意 | 一键运行整套系统 |

> **如果你只是想看效果（不改代码）**：上面只装 Docker 即可，跳到 [02-启动指南](./02-启动指南.md) 的 A 方案。
> **如果你要改代码**：上面除 Docker 外都要装。

## 1. JDK 17

### Windows

1. 打开 <https://www.injdk.cn/> 或 <https://adoptium.net/zh-CN/>
2. 下载 **Eclipse Temurin 17 LTS** Windows x64 `.msi` 安装包
3. 双击安装，**勾选「Set JAVA_HOME variable」与「Add to PATH」**
4. 装完打开 PowerShell 或 CMD，执行：

```cmd
java -version
```

应看到类似（`17.x.x` 是关键）：

```
openjdk version "17.0.10" 2024-01-16
OpenJDK Runtime Environment Temurin-17.0.10+7
```

### macOS

```bash
# 推荐用 Homebrew
brew install --cask temurin@17

# 或者用 SDKMAN
curl -s "https://get.sdkman.io" | bash
sdk install java 17.0.10-tem
```

验证：`java -version` 应显示 17。

### Linux（Ubuntu/Debian）

```bash
sudo apt update
sudo apt install -y openjdk-17-jdk
java -version
```

### 常见问题

| 现象 | 原因 / 解决 |
| --- | --- |
| `java: command not found` | PATH 没配好。Windows 重开终端；Linux/Mac 把 `JAVA_HOME/bin` 加到 PATH |
| `java -version` 显示 8 或 11 | 装了多个 JDK。Windows 控制面板→系统属性→环境变量；Linux `sudo update-alternatives --config java` |
| 装的是 Oracle JDK 还是 OpenJDK？ | 都行。本项目兼容所有 JDK 17 发行版（Temurin / Zulu / Oracle / Microsoft） |

## 2. Node.js + pnpm

### Windows

1. 打开 <https://nodejs.org/zh-cn/> 下载 **LTS 版本**（20.x）
2. 双击安装，全部默认即可
3. 验证：

```cmd
node -v   # 应输出 v20.x.x
npm -v    # 应输出 10.x.x
```

4. 装 pnpm：

```cmd
npm install -g pnpm
pnpm -v   # 应输出 9.x.x
```

### macOS / Linux

```bash
# macOS
brew install node@20
npm install -g pnpm

# Linux 用 nvm（推荐）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install 20
npm install -g pnpm
```

### 配置国内 npm 镜像（项目内已配，可跳过）

项目里 `frontend/.npmrc` 已经把 registry 指向 npmmirror，不用额外配。如果你想全局也用：

```bash
pnpm config set registry https://registry.npmmirror.com
```

## 3. MySQL 8

### Windows

最简单的方式：装 **MySQL Installer**

1. 下载 <https://dev.mysql.com/downloads/installer/>
2. 选 "MySQL Installer Community"，下载小包（web）即可
3. 安装时选 **Server only**，端口 `3306`，root 密码设为 `root`（教学环境，方便统一）
4. 装完确认服务正在运行（任务管理器 → 服务 → 找 `MySQL80`）

### macOS

```bash
brew install mysql@8.0
brew services start mysql@8.0
mysql -u root  # 默认无密码，进去后改密码
```

进入 mysql 后：

```sql
ALTER USER 'root'@'localhost' IDENTIFIED BY 'root';
FLUSH PRIVILEGES;
```

### Linux（Ubuntu）

```bash
sudo apt install mysql-server
sudo mysql_secure_installation   # 按提示设置 root 密码为 root
```

### 验证

```bash
mysql -uroot -proot -e "SELECT VERSION();"
```

应该看到 `8.0.xx` 字样。

### 创建项目数据库

```bash
mysql -uroot -proot
```

进入 MySQL 命令行后：

```sql
CREATE DATABASE his DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
SHOW DATABASES;        -- 应能看到 his
EXIT;
```

> **如果你的 root 密码不是 `root`**：修改 `backend/src/main/resources/application-dev.yaml` 里的 `password` 字段，或把 MySQL 密码改回 `root`（教学环境推荐统一）。

## 4. Git

### Windows

下载 <https://git-scm.com/download/win>，全部默认安装。

### macOS

```bash
brew install git
# 或：xcode-select --install
```

### Linux

```bash
sudo apt install -y git
```

### 必做的初始化

```bash
git config --global user.name  "你的姓名"
git config --global user.email "你的邮箱"
```

### 配置 SSH 密钥（推荐，避免每次 push 输密码）

```bash
ssh-keygen -t ed25519 -C "你的邮箱"
# 一路回车

# 把公钥复制到剪贴板
# Windows: clip < ~/.ssh/id_ed25519.pub
# macOS:   pbcopy < ~/.ssh/id_ed25519.pub
# Linux:   cat ~/.ssh/id_ed25519.pub  # 手动复制
```

然后：GitHub 头像 → Settings → SSH and GPG keys → New SSH key → 粘贴 → Add。

测试：

```bash
ssh -T git@github.com
# 应看到：Hi <你的用户名>! You've successfully authenticated, ...
```

## 5. IDE：IntelliJ IDEA（后端）

1. 下载 <https://www.jetbrains.com/idea/>，选 **Community Edition**（免费够用）
2. 装完打开，第一次启动会让选主题、插件等，**默认即可**
3. 推荐安装的插件（File → Settings → Plugins）：
   - **Lombok**（很可能已经装好）
   - **MyBatis Log**（可选，让 SQL 日志直接成可执行 SQL）
   - **Translation**（可选，翻译注释和文档）
4. 打开本项目：`File → Open` → 选 `his/backend` 目录
5. IDEA 会自动识别 Gradle 项目并下载依赖（约 3–5 分钟）
6. 等右下角进度条跑完，左侧就能看到完整的项目树

详细的 IDE 配置和调试技巧见 [10-IDE-与调试-Git.md](./10-IDE-与调试-Git.md)。

## 6. IDE：VS Code（前端）

1. 下载 <https://code.visualstudio.com/>，全部默认安装
2. 装完打开，左侧扩展商店（图标像四个方块）搜索并安装：
   - **Vue (Official)** —— Vue 3 官方插件
   - **ESLint** —— 代码风格检查
   - **GitLens**（可选） —— 增强的 Git
   - **Chinese (Simplified) Language Pack** —— 简体中文界面
3. 打开本项目：`File → Open Folder` → 选 `his/frontend` 目录
4. 按 `` Ctrl+` ``（反引号）打开终端，运行 `pnpm install` 安装依赖（约 2 分钟）

## 7. Docker Desktop（可选）

只有走 [02-启动指南](./02-启动指南.md) 的 A/B 方案才需要。

1. 下载 <https://www.docker.com/products/docker-desktop/>
2. Windows 用户：安装后可能要重启，启动后会让你打开 WSL2 集成（默认即可）
3. 验证：

```bash
docker --version           # 应输出 24.x 或更高
docker compose version     # 应输出 v2.x
```

> **国内拉镜像很慢的处理**：见 [05-国内网络优化.md](./05-国内网络优化.md)。

## 8. 拉取项目代码

```bash
# 在你想放项目的目录里
git clone git@github.com:hmc-edu/his.git
cd his

# 看看目录结构
ls   # 应能看到 backend/ frontend/ docs/ docker-compose.yml 等
```

如果 `git clone` 慢/失败，先按上面 SSH 那步配好密钥；或者临时改用 HTTPS：

```bash
git clone https://github.com/hmc-edu/his.git
```

## 9. 环境检查清单

下面命令全部能跑通，就说明环境装好了：

```bash
java -version           # 17.x.x
node -v                 # v20.x.x（或 18.x）
pnpm -v                 # 9.x.x
mysql --version         # 8.0.x
git --version           # 2.x
docker --version        # 24.x（可选）
```

打开项目检查 MySQL 库存在：

```bash
mysql -uroot -proot -e "SHOW DATABASES;" | grep his
# 应输出：his
```

全部通过就可以进入 [02-启动指南.md](./02-启动指南.md) 把后端、前端跑起来了。
