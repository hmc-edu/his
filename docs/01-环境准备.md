# 01 · 环境准备

## 必备软件

| 软件 | 版本 | 用途 | 验证命令 |
| --- | --- | --- | --- |
| JDK | 17+ | 运行后端 | `java -version` |
| Node.js | 18+ | 运行前端 | `node -v` |
| pnpm（或 npm/yarn） | 任意 | 前端包管理 | `pnpm -v` |
| MySQL | 8.0+ | 数据库 | `mysql --version` |
| Git | 2.x | 版本控制 | `git --version` |

> Gradle 不需要单独安装：项目自带 `gradlew`（Gradle Wrapper），首次运行会自动下载 Gradle 8.5。

## 推荐 IDE

- 后端：IntelliJ IDEA（社区版即可）
- 前端：VS Code，建议安装插件 `Vue (Official)`、`ESLint`

## 数据库初始化

```sql
-- 1. 创建数据库
CREATE DATABASE his DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 2. 进入数据库
USE his;

-- 3. 执行建表脚本
SOURCE /path/to/his/backend/src/main/resources/db/schema.sql;

-- 4. 导入演示数据（科室/药品/患者）
SOURCE /path/to/his/backend/src/main/resources/db/data.sql;
```

> 系统默认账号 `admin / reception / doctor`（密码 `123456`）以及示例医生记录由后端启动时的 `DataInitializer` 自动写入，无需手工导入。

## 修改数据库连接

如果你的 MySQL 用户名/密码或地址不同，修改 `backend/src/main/resources/application-dev.yaml`：

```yaml
spring:
  datasource:
    username: root
    password: 你的密码
    url: jdbc:mysql://localhost:3306/his?useUnicode=true&characterEncoding=utf8&...
```
